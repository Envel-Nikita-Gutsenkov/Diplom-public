# Решение проблем (Troubleshooting Guide)

Данный документ содержит ответы на часто возникающие технические вопросы и способы решения типовых ошибок при работе с Diplom-public.

## Аутентификация и Доступ

### Где взять данные для входа администратора?

При первом запуске через `start.sh` или `seed` скрипт создается учетная запись администратора:
- **Email**: `admin@admin.com`
- **Пароль**: `admin`

### Ошибка: "Invalid Credentials"
Если вы уверены в правильности пароля, но войти не удается:
1. Убедитесь, что миграции БД применены корректно (`npx prisma db push`).
2. Проверьте переменную `AUTH_SECRET` в `.env` — она не должна быть пустой.

## Инфраструктура и Docker

### Контейнеры не запускаются
Выполните `docker-compose logs -f` для просмотра ошибок в реальном времени.
- **Проблема с портами**: Если порт 3000 или 5432 занят, измените привязку портов в `docker-compose.yml`.
- **Проблема с правами**: Убедитесь, что у Docker есть права на запись в директории `backups/` и `prisma/`.

### База данных недоступна (P1001)
- Убедитесь, что контейнер `db` запущен.
- Проверьте `DATABASE_URL` — в production он должен указывать на хост `db`, а не на `localhost`.

## Разработка и Тесты

### Ошибки при установке пакетов
Если после обновления `package.json` возникают конфликты:
```bash
rm -rf node_modules .next package-lock.json
npm install
```

### Тесты Playwright падают в Docker
Убедитесь, что контейнер приложения успел запуститься до начала тестов. Скрипт `start.sh` содержит встроенную проверку healthcheck, но в некоторых случаях может потребоваться ручной перезапуск контейнера `e2e`:
```bash
docker-compose restart e2e
```

## Системное обслуживание

### Как сбросить базу данных?
**Внимание: Все данные будут удалены!**
```bash
npx prisma db push --force-reset
```

### Как просмотреть логи приложения?
```bash
docker-compose logs -f app
```

## Python и Pyodide

### Ошибка загрузки Pyodide (Network Error)
- Проверьте доступ к `https://cdn.jsdelivr.net`.
- Если возникает ошибка CSP, убедитесь, что в `next.config.ts` в `contentSecurityPolicy` добавлены разрешения для `script-src` и `connect-src`.
- Используется стабильная версия Pyodide **v0.29.3**. Не рекомендуется обновлять до v0.29.4 без проверки доступности файлов на CDN.

### Код завис или выдает Out of Memory
- Приложение использует Web Workers. Если скрипт участника (например, `while True`) заблокирует поток, используйте кнопку **«Остановить»** для принудительного завершения воркера.
- Лимит вывода консоли — 1000 строк. При превышении старые строки удаляются.
