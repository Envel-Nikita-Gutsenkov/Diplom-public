# Дипломный Проект: Система Организации Олимпиад

Современное веб-приложение для проведения олимпиад, разработанное с использованием передовых технологий веб-разработки. Проект ориентирован на высокую производительность, строгую безопасность и интуитивно понятный пользовательский интерфейс.

## Ключевые Особенности

- **Fullstack Архитектура**: Построено на Next.js 15+ с использованием App Router и Server Actions.
- **Python-песочница**: Встроенная среда выполнения кода (Pyodide) с поддержкой библиотек (numpy, pandas и др.) и защитой от бесконечных циклов.
- **Производительность**: Использование React 19 и нативного хеширования паролей (bcrypt).
- **Безопасность**: Полная изоляция базы данных PostgreSQL внутри Docker-сети, защита роутов через Middleware.
- **Умное сохранение**: Автоматическое сохранение ответов с восстановлением состояния при перезагрузке страницы.
- **Интерактивность**: Плавные анимации на базе Framer Motion и современный дизайн с использованием Shadcn UI (Tailwind CSS v4).
- **Контейнеризация**: Полная готовность к развертыванию через Docker Compose ("Запустил и забыл").
- **Тестирование**: Кодовая база покрыта Unit-тестами (Jest) и E2E тестами (Playwright).

## Быстрый старт (Production на сервере)

Для полного автоматического развертывания на чистом сервере Ubuntu выполните:

```bash
# 1. Склонируйте репозиторий
git clone https://github.com/Envel-Nikita-Gutsenkov/Diplom-public.git
cd DiplomSoft2

# 2. Запустите единый скрипт настройки
# Скрипт сам создаст .env, выпустит SSL через Let's Encrypt и настроит Nginx
chmod +x setup-prod.sh
sudo ./setup-prod.sh
```

**Администратор по умолчанию:** `admin@admin.com` / `admin`
